## netselect

A netselect CLI Tool

### Synopsis

netselect is an open source CLI tool to select the fastest host based on the lowest ICMP latency.

```
netselect [flags] <host(s)>
```

### Examples

```
$ netselect m1.example.com m2.example.com m3.example.com
$ netselect -v

```
